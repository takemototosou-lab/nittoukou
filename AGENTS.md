# プロジェクト概要

日本ペイント（日塗工）のRGB値を、
白・黒・赤・黄・青の5原色比率へ変換する
塗装職人向けAIアプリ。

## 目的

完全一致調色ではない。

現場で調色を始めるための
「初期比率」を出すことが目的。

## 基本仕様

入力：
- 日塗工番号
- RGB値

出力：
- 白 %
- 黒 %
- 赤 %
- 黄 %
- 青 %

合計100%

## 重要

現場職人向けなので、
理論精度より
「方向性」を重視する。

## UI方針

- スマホ優先
- シンプル
- 数字を大きく
- 入力最小

## ロジック方針

- 明色 → 白増加
- 暗色 → 黒増加
- R優勢 → 赤増加
- R+G優勢 → 黄増加
- B優勢 → 青増加

## 注意

完璧な色再現を目指さない。
調色開始地点を作るAI。
